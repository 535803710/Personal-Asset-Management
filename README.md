# Personal-Asset-Management

基于飞书多维表格（Feishu Base）打造的**全维度个人资产与订阅管理系统**。不仅关注软件订阅续费，更引入了**资产负债管理**与**日均成本（Amortization）**分析思维，帮助你算清每一笔消费的真实价值。

## 📖 项目简介 (Introduction)

本项目是一个可视化的个人财务健康仪表盘。它突破了传统记账软件的流水账模式，采用“资产全生命周期”的管理视角：
*   对于**软件与服务**，关注“每日持有成本”与“续费倒计时”。
*   对于**固定资产**（房/车/数码），关注“资产净值”、“负债进度”与“使用回本率”。

通过飞书强大的仪表盘功能，让你对自己的 **资产总值 (Total Assets)**、**负债总额 (Total Liabilities)** 以及 **每日生存成本 (Daily Burn Rate)** 一目了然。

## ✨ 核心功能 (Key Features)

### 1. 软件与服务订阅管理 (Software & Services)
不仅仅是软件，也包含物业、停车等周期性服务。
*   **💰 每日成本计算**：自动将年付/月付金额转化为“每日成本”（如：某软件年费 365元 = 每天 1元），直观感受消费压力。
*   **⏳ 智能续费倒计时**：首页醒目展示距离下一次扣费的天数，支持“续费预警”排行。
*   **📊 支出结构分析**：
    *   **软件 vs 服务**：区分纯数字产品订阅与生活服务支出。
    *   **使用时长占比**：分析哪些服务陪伴你的时间最久。

### 2. 固定资产与负债管理 (Assets & Liabilities)
覆盖房产、车辆、3C数码、家用电器等大额资产。
*   **🏠 房贷/车贷追踪**：
    *   可视化展示**还款进度**（如：房贷进度 65.71%）。
    *   实时计算**剩余还款期数**与**剩余还款金额**。
*   **📉 日均使用成本 (Amortization)**：
    *   计算资产的“日均使用成本”（购买金额 / 已使用天数）。
    *   **“耐用王”排行**：通过使用时长排名，通过数据告诉你哪些东西买得最值（用得越久，日均成本越低）。
*   **🗂️ 精细化分类**：
    *   **3C数码**：手机、电脑、iPad。
    *   **家用电器**：新风、净水器、扫地机。
    *   **大宗资产**：房产、车辆。

### 3. 🤖 自动化工作流 (Automation Workflows)
利用飞书多维表格的“自动化流程”功能，实现零人工干预的资产状态更新。

#### 🏠 房产/大额资产自动更新
告别每月手动更新余额的繁琐。系统会在还款日**自动扣减剩余贷款本金与期数**，实时驱动仪表盘上的“还款进度条”向前推进，让你随时掌握最新的资产净值与负债情况。

#### 🔔 订阅智能到期提醒
化被动为主动。在软件或服务续费前（如提前 3 天），系统会通过飞书机器人**主动推送消息卡片**。这不仅是提醒，更是给你留出“是否继续订阅”的决策缓冲期，彻底杜绝“忘记取消而被自动扣费”的烦恼。

## 🛠 技术/数据架构 (Architecture)

*   **核心数据库**：飞书多维表格 (Feishu Base)
*   **数据表设计**：
    *   **软件管理表**：名称、金额、购买天数、每日成本、到期时间、剩余天数、类型（软件/服务）。
    *   **资产管理表**：名称、购买金额、开始使用时间、已用天数、每日使用成本、类型、剩余贷款、剩余还款期数。
*   **自动化逻辑**：自动化流程触发到期通知与数据更新。

## 🚀 快速开始 (Getting Started)

![](./image/show.gif)

### 1. 获取模版
*  - 下载模版文件（📱个人资产管理脱敏分享版本）
*  - 打开飞书文档
*  - 点击左上角"导入"按钮
*  - 选择导入的模版文件

### 2. 录入数据
*   **软件类**：输入名称、价格、周期（年/月），系统自动计算每日成本。
*   **资产类**：输入总价、购买日期；若有贷款，输入贷款总额与已还期数，系统自动生成负债图表。

### 3. 配置自动化
*   进入多维表格右上角“自动化”面板。
*   启用预设的 **“房产自动计算”** 和 **“软件到期提醒”** 流程。
*   根据实际还款日修改触发时间。

## 💡 使用场景 (Use Cases)

- **场景一：订阅大扫除**
    - 查看“软件每日成本”排行，发现某视频会员每天成本 0.8元但几乎不看，决定取消续费。
- **场景二：房贷激励**
    - 每月还款日后，自动化流程自动跑通，看着“房贷进度条”从 53% 自动涨到 54%，直观看到负债减少，增加搞钱动力。
- **场景三：理性消费决策**
    - 想买新 iPad 前，看一眼旧 iPad 的“日均使用成本”仅为 1.74元/天，且已使用 3000+ 天，确认它是高性价比生产力工具，放心升级。
